# 03｜可达成的自动化（Automation）

## 1) 低风险自动化（默认开启）
- 语音/图片 → 自动归档到 Airtable/Notion/Obsidian（可选）
- 自动生成摘要（1句话 + 3要点）
- 自动生成“下一步动作建议”（不直接执行）

## 2) 中风险自动化（需要确认）
- 识别日期/期限 → 创建提醒（到期前 90/30/7 天）
- 会议/任务 → 自动草拟邮件/消息（你点发送）

## 3) 高风险自动化（强治理）
- 直接在系统里执行（改数据/发通知/提交工单）
- 需要：RBAC + 审计 + 回滚 + 白名单出网

## 4) 与 OpenClaw 的结合
- OpenClaw 负责：记忆、工具调用、任务编排、跨端交互
- 端侧器官负责：输入捕获、低摩擦触发、上下文沉淀
