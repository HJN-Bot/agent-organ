# 01｜概念总结（从“硬件器官”出发）

## 1. 一句话定义
Agent Organ 是一个“可穿戴的 Agentic 器官”，把你在生活/工作中产生的碎片信息低摩擦捕获，并通过长期记忆与自动化编排，持续为你减少认知负担与执行成本。

## 2. 为什么是“器官”而不是“设备”
- 器官强调：**持续、自然、低摩擦**（不需要你每次打开App）
- 器官强调：输入的首要价值是“感知”，不是“炫技”
- 器官强调：输出要像神经反射一样——少打扰、但关键时刻很有用

## 3. 典型使用场景（优先级）
1) 随手记：语音 10 秒 → 自动变成 Task/Idea/会议纪要
2) 随手看：拍照/截图 → OCR → 自动归档为 Evidence（证据）
3) 随手提醒：识别“日期/期限/风险点” → 自动提醒
4) 随手总结：一天的碎片输入 → 晚间复盘摘要

## 4. 成功指标（产品层）
- 捕获摩擦：1 秒内可开始记录
- 结构化准确：任务/想法分类准确率、关键信息抽取准确率
- 打扰控制：提醒命中率高、误报少（可配置）
- 可审计：所有自动化动作可追溯（像医疗器械/合规系统）
