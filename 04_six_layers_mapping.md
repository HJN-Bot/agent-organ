# 04｜映射到“六层框架”（OpenClaw/R·Agent 的软件层打通）

用你之前认可的六层来描述 Agent Organ（硬件+软件一体）：

## L1 交互层（入口）
- 端侧：按钮/唤醒词/触摸/拍照
- 手机：快捷指令/通知入口
- 桌面：Dashboard/Review UI

## L2 工作流层（编排）
- LangGraph/Temporal/n8n：把“捕获→结构化→入库→提醒/执行”做成可暂停/可重跑的状态机

## L3 工具层（能力）
- ASR、OCR、分类器、RAG检索、写入 Airtable/DB、提醒发送

## L4 记忆层（资产）
- Job：一次事件处理上下文
- Project：个人偏好/常用模板/术语表
- Org：组织规范（如果走企业/医疗器械方向）

## L5 元数据层（追溯）
- 每个 event 的输入/输出/引用/模型版本/成本
- 便于质量回归与审计

## L6 治理层（安全/合规）
- 最小权限、密钥管理、出网白名单
- 审计日志、备份、错误回放
- 对外可类比“医疗器械软件”的质量体系
